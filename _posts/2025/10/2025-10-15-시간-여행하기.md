## 프로젝트의 변경사항들을 타임캡슐(버전)에 담기

### 변경 사항 확인

```bash
git status
```

### 파일 하나 담기

```bash
git add file_name
```

### 모든 파일 담기

```bash
git add .
```

## 타임캡슐 묻기

```bash
git commit -m "커밋 메세지"
```

### add와 commit 한번에

```bash
git commit -am "커밋 메세지"

- 새로 추가된(untracked) 파일이 없을 때 한정
```

### 로그 확인

```bash
git log
```

## 과거로 돌아가는 두 방식

### reset

- 원하는 시점으로 돌아간 뒤 이후 내역들을 지움

```bash
git reset --hard 커밋해시코드
```

### revert

- 되돌리기 원하는 시점의 커밋을 거꾸로 실행

```bash
git revert 커밋해시코드
```
