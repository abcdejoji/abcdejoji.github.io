## 커밋하지 않은 변경사항 되돌리기

```bash
git restore (파일명)
```

### 변경상태를 스슽메이지에서 워킹 디렉토리로 돌려놓기

```bash
git restore --staged (파일명)
```

### 파일을 특정 커밋의 상태로 되돌리기

```bash
git restore --source=(커밋 해시) 파일명
```

## reset 한걸 다시 되돌리기

```bash
git reflog
```

- 프로젝트가 위치한 커밋이 바뀔때마다 기록되는 내역을 보여줌
- 이 내역을 통해 `reset` 전으로 복구 할 수 있음
